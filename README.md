# CSV_IO

## Get the code:

Using GIT:
``` bash
git clone https://github.com/r-asif/CSV_IO.git
```

Or download the files:

https://github.com/r-asif/CSV_IO/archive/master.zip

## readCSV
Reads a CSV with schema (ID,Make,Color)
``` bash
python3 readCSV.py
```

## writeCSV
Writes a CSV 
``` bash
python3 writeCSV.py
```
